■前提条件
・ODBCの設定が必須です。
DB2Clientのインストールが済んでおり、ODBC経由で接続できることを確認してください。

・Windows10で動作確認しています。
Windows7ではPowershellのバージョンが違うため動作しない可能性が高いです。
Windows7で使用する場合は、Powershellを5.0以上にバージョンアップしてください。

・ローカルにコピーして使用してください。
Powershellは標準でリモートに保存されているスクリプトは実行できません。
デスクトップなどローカルにコピーしてから実行してください。

・階層が深すぎて処理が終わらなかったため、最大で8階層まで調査するように制限しています。
